### 👨‍💻 Về bản thân

* 🎓 Sinh viên ngành Công nghệ thông tin / Khoa học máy tính tại **PTIT**.
* 🧠 Đang đào sâu nghiên cứu về **Artificial Intelligence**, đặc biệt chú trọng vào NLP (RAG, BERT, Word2vec) và Computer Vision (YOLO, CNNs, R-CNN).
* ⛓️ Đam mê phát triển Web3, làm việc với Smart Contracts (Solidity), tích hợp IPFS và xây dựng các hệ thống dựa trên NFT (ERC721).
* 🛠️ Xây dựng các hệ thống Backend vững chắc với **Java** (Swing, JDBC) và **Python** (Flask, FastAPI) cùng kiến trúc CSDL chuẩn hóa (MySQL, SQL Server).

### 🚀 Dự án nổi bật

* 🍃 **FoodSaver:** Hệ sinh thái ứng dụng di động tích hợp AI giúp dự đoán thời gian hư hỏng của thực phẩm, hướng tới mục tiêu giảm thiểu lãng phí thức ăn trong hộ gia đình.
* 📚 **Library Blockchain Kit:** Hệ thống quản lý thư viện phi tập trung, xử lý logic mượn/trả sách thông qua NFT, được deploy trên mạng thử nghiệm Sepolia.
* 🏫 **Student & Asset Management System:** Ứng dụng Full-stack kết hợp Backend Java với Frontend React/Next.js, áp dụng các chuẩn thiết kế kiến trúc và CRUD logic tối ưu.
